# Challenge-I Data-Science -  Optimización de Red de Tiendas

**1. Descripción del Proyecto**
Este proyecto analiza el rendimiento operativo y financiero de cuatro tiendas comerciales con el objetivo de identificar oportunidades de optimización. A través de técnicas de análisis de datos con Pandas y Matplotlib, evaluamos métricas clave (KPIs) como ingresos totales, rotación de productos por categoría, satisfacción del cliente y eficiencia logística.

**2. **Metodología de Análisis****
El análisis siguió el ciclo de vida estándar de ciencia de datos:

**Consolidación de Datos:** Integración de los datasets de las 4 tiendas.

**Limpieza y Procesamiento:** Uso de funciones como .value_counts(), .sum() y .mean() para estructurar la información.

**Exploración Estadística:** Identificación de patrones mediante diagramas de caja (Boxplots) y gráficos de dispersión.

**Visualización:** Implementación de gráficos de barras comparativos y análisis de distribución.

**3. Principales Hallazgos**
**Rendimiento Financiero:** La Tienda 1 destaca como el activo con mayores ingresos, mientras que la Tienda 4 muestra un déficit constante.

**Brecha en Ventas:** La Tienda 4 presenta una caída crítica en categorías de alto valor (Electrodomésticos y Deportes).

**Eficiencia Logística:** Identificamos que la Tienda 4 tiene los costos de envío más bajos, lo cual no ha sido suficiente para mejorar su competitividad comercial.

**4. Conclusión y Recomendación**
Tras el análisis integral, se recomienda al Sr. Juan proceder con la venta de la Tienda 4. La combinación de bajos ingresos, baja rotación de productos y una experiencia de usuario (calificación) inferior, la posicionan como un pasivo para la rentabilidad general del negocio.

**5. Tecnologías Utilizadas**
**Lenguaje:** Python 3.x

**Librerías:** 
**Pandas:** Manipulación y análisis de datos.

**Matplotlib:** Visualización de datos.

numpy: Cálculo numérico.
